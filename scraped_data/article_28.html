What is a reference ID or UTR Number ( Unique Transaction Reference Number)


This is a number that is unique to every IMPS/ NEFT or RTGS transaction you do and helps us match that transaction to you.



Where do I get this number from?


This number is given by your bank itself while processing your funds.


IMPS | UPI : This is a 12 digit number 

NEFT / RTGS : This is an alphanumeric number which normally starts with your bank’s Code


This will show up in your bank statement

Points to remember

  • Do Not change the format or edit any part of this number

Always remember to quote this number  for any fiat transactions when raising a ticket with us.
